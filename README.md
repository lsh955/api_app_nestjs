## Description

[Nest](https://github.com/nestjs/nest) 프레임워크와 TypeScript 테스트 목적으로 시작한 회원가입, 이메일인증, 로그인, 회원정보조회 등으로 구성한 유저서비스

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development watch mode
$ yarn run start:dev

# staging mode
$ yarn run start:stg

# production mode
$ yarn run start:prod
```
